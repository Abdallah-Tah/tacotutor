# Instruction: Memorization Mode

## Metadata
level: intermediate
mode: memorization
pace: medium
subject: quran

## Target
- Surah: Al-Ikhlas
- Ayah: 1-4

## Goals
- Memorize the full surah
- Recite without looking at text
- Understand the meaning of each ayah

## Teaching Rules
- Show full ayah first, then hide it
- Ask child to recite from memory
- Provide hints if child gets stuck (first letter, last word)
- Break into smaller chunks if needed
- Review previous ayahs before learning new ones

## Correction Rules
- If child makes a mistake, show the correct ayah again
- Ask them to read it twice, then try from memory again
- Track which ayahs are memorized and which need review
- Be patient - memorization takes time

## Visual Guidance Rules
- Show ayah text clearly for initial learning
- Hide text during memorization attempts
- Show hints (first letter) if child struggles
- Use progress bar to show memorization status

## Tutor Behavior
- Be encouraging throughout
- Celebrate each ayah memorized
- Review all memorized ayahs at end of session
- Track which ayahs need review in next session

## Parent Notes
- Daily review is key for memorization
- 15-20 minutes per session
- Encourage child to practice between sessions
- Praise memorization milestones
